8.12

成就：完成Essential C++的第二章

收获与心得：

- 提领指针之前，一定要盘断是否为空指针
- C++没有提供任何可以在`new`数组的同时对数组进行初始化
- `new`创建的用`delete`销毁，`new[]`创建的用`delete[]`销毁
- 函数参数使用**引用**时，引用的值不能是空(nullptr或0)
- 函数使用默认参数时，参数默认值只能说明一次，且要在函数声明处指定
- 重载函数不能以返回值的类型区分，因为调用时无法选择
- 重载函数中对变量的处理只有类型不同时，考虑模板函数
- **inline函数的定义必须在头文件完成**
